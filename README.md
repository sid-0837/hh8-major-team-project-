# 🚨 AI-Based SIEM Tool

## 📌 Overview
This project is an AI-based Security Information and Event Management (SIEM) system designed to detect and analyze security threats in real-time using both rule-based and machine learning techniques.

---

## 🎯 Features

- 🔍 Log Ingestion System
- ⚠️ Multi-Attack Detection:
  - Brute Force Attack
  - SQL Injection Attack
  - Unauthorized Access
- 🤖 Machine Learning-based Anomaly Detection
- 📊 Risk Scoring System
- 📈 Real-Time Dashboard (Table + Graph)
- 🗄️ PostgreSQL Database Integration

---

## 🛠️ Tech Stack

- Backend: FastAPI (Python)
- Database: PostgreSQL
- Frontend: HTML, CSS, JavaScript
- Visualization: Chart.js
